Description

Erlang http status code library

this library provides simple methods to get the description or reference to an http status code.

The list is taken from: http://www.iana.org/assignments/http-status-codes/http-status-codes.xhtml

Data Types

httpStatusCode()

httpStatusCode() = 100..599

httpStatusDescription()

httpStatusDescription() = bitstring()

httpStatusReference()

httpStatusReference() = bitstring()
